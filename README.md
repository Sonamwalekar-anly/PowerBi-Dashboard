# 📊 Power BI Dashboard Project

## 📌 Project Overview

This project presents an interactive Power BI dashboard built to analyze and visualize business data. The dashboard transforms raw data into meaningful insights through dynamic charts, KPIs, filters, and drill-down capabilities, enabling data-driven decision-making.

---

## 🎯 Objective

The objective of this project is to:

- Analyze business performance using Power BI
- Create interactive visualizations
- Track important KPIs
- Identify trends and patterns
- Support data-driven business decisions

## ⚡ DAX Measures Used

Examples:

```DAX
Total Sales = SUM(Sales[Sales Amount])

Total Profit = SUM(Sales[Profit])

Profit Margin =
DIVIDE([Total Profit],[Total Sales])

Total Orders =
DISTINCTCOUNT(Sales[Order ID])
```
